# 📖 About

Apache Kafka is one of the most widely used distributed event streaming platforms in modern Data Engineering.

This repository documents my hands-on learning journey while implementing Kafka concepts through practical examples and Docker-based environments.

The focus is on understanding how real-time data flows through producers, brokers, topics, partitions, and consumers.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Vinay9222/kafka-for-data-engineers.git

cd kafka-for-data-engineers
```

---

## Start Kafka

```bash
docker compose up -d
```

---

## Stop Kafka

```bash
docker compose down
```
